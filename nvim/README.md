# Neovim Config

Personal Neovim configuration built on [Packer](https://github.com/wbthomason/packer.nvim) with LSP, Treesitter, Telescope, and a VS Code-inspired dark theme.

## Prerequisites

- [Neovim](https://neovim.io/) >= 0.8
- [Git](https://git-scm.com/)
- A [Nerd Font](https://www.nerdfonts.com/) (optional, for icons)
- [ripgrep](https://github.com/BurntSushi/ripgrep) (for Telescope live grep)
- [tmux](https://github.com/tmux/tmux) (optional, for `tmux-sessionizer` integration)

## Installation

### 1. Back up existing config (if any)

```sh
mv ~/.config/nvim ~/.config/nvim.bak
```

### 2. Clone the repo

```sh
git clone https://github.com/cherniaky/neovim-config.git ~/.config/nvim
```

### 3. Install Packer

```sh
git clone --depth 1 https://github.com/wbthomason/packer.nvim \
  ~/.local/share/nvim/site/pack/packer/start/packer.nvim
```

### 4. Install plugins

Open Neovim and run:

```
:PackerSync
```

Wait for all plugins to install, then restart Neovim.

### 5. Install LSP servers

LSP servers are managed by [Mason](https://github.com/williamboman/mason.nvim). Open Neovim and run:

```
:Mason
```

Install the language servers you need (e.g. `lua_ls`, `ts_ls`, `rust_analyzer`).

## Structure

```
~/.config/nvim/
├── init.lua                        # Entry point — loads lua/yuriicherniak/
├── lua/yuriicherniak/
│   ├── init.lua                    # Loads packer, remap, and set modules
│   ├── packer.lua                  # Plugin declarations
│   ├── remap.lua                   # Global key mappings
│   └── set.lua                     # Vim options (line numbers, tabs, etc.)
├── after/plugin/                   # Plugin-specific configs (loaded after plugins)
│   ├── colors.lua                  # VS Code dark theme
│   ├── fugitive.lua                # Git integration keymaps
│   ├── gitsigns.lua                # Git gutter signs & hunk actions
│   ├── harpoon.lua                 # Quick file navigation
│   ├── lsp.lua                     # LSP, completion, and diagnostics
│   ├── telescope.lua               # Fuzzy finder keymaps
│   ├── treesitter.lua              # Syntax highlighting
│   └── undotree.lua                # Undo history visualizer
├── ftplugin/
│   └── java.lua                    # Java-specific LSP config (jdtls)
└── plugin/
    └── packer_compiled.lua         # Auto-generated by Packer (do not edit)
```

## Editor Settings

- **Line numbers:** Absolute + relative
- **Indentation:** 4 spaces, smart indent, no tabs
- **Word wrap:** Off
- **Search:** Highlight + incremental search
- **Undo:** Persistent undo (saved to `~/.vim/undodir`)
- **Cursor:** Thin vertical bar in insert mode
- **Scroll offset:** 10 lines
- **Color support:** 24-bit true color enabled
- **Swap/backup files:** Disabled

## Updating

Pull the latest changes and sync plugins:

```sh
cd ~/.config/nvim
git pull
nvim -c 'PackerSync'
```

To update LSP servers, open Neovim and run `:Mason`, then press `U` to update all.

## Uninstall

```sh
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.cache/nvim
```
